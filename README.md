# The Nathwani Commission

The Nathwani Commission was a 1979 independent inquiry appointed by the Citizens for Democracy (a prominent Indian human rights NGO) to investigate alleged human rights abuses and the infringement of civil liberties within the Dawoodi Bohra community by their religious head, the Syedna.

The original report (available in `./source-material`) was published in Urdu. This project endeavours to produce an accurate English translation.

## Source Material
The Urdu source text was obtained from [Rekhta](https://www.rekhta.org/ebooks/detail/nathwani-commission-report-dawoodi-buhara-commission-n-p-nathwani-ebooks).

## Translation Methodology
The bulk of the translation was done using Claude Opus 4.8. Each chapter was translated one at a time with the following project description and instructions provided to the model (plus a simple "Translate Chapter X" prompt).

```
Project Description
I want to translate the Nathwani Commission Report of 1979, an Urdu document into English, using Typst for typesetting in the interest of preserving formatting. I have already created a Typst project with boilerplate for the translation and two chapters already translated. I would like to proceed chapter by chapter in continuing the translation, until all 14 chapters plus appendices are translated

Project Instructions
Each chapter is uploaded in a separate PDF. For each chapter in the report, I will prompt you with the chapter name. You should read the corresponding PDF file, translate that chapter carefully and produce a corresponding Typst document that renders an English translation in similar formatting as the source. I should be able to add the document to my project and directly include it, just like with chapters 1 and 2. 

Avoid additional commentary during the translation process; I just want the finished result and some final comments afterwards if important. Do not be afraid to skip past certain words/phrases/citations if they are difficult to read or translate. In such cases, simply leave an annotation in the translation and move on. Write the Typst document as you proceed with the translation, rather than constructing it all at once, so that if you get interrupted you can easily pick up where you left off.
```

The translation work is ongoing. The next steps are to go through the translation carefully and correct any errors made by the model, clean up the typesetting, fill in any gaps in the translation, etc.
